# capsule_networks
Implementation of Dynamic Routing Between Capsules (http://papers.nips.cc/paper/6975-dynamic-routing-between-capsules.pdf)

# Result

CapsNet with 3 routing and reconstruction: 99.7% accuracy on validation set.
